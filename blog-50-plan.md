## File Structure
Each blog post creates:
```
src/app/(public)/blog/{slug}/page.tsx
```

## Template Pattern (per post)
- Next.js `Metadata` export (title, description, OpenGraph, Twitter, canonical, robots)
- JSON-LD structured data (`@graph` with Article + FAQPage)
- Breadcrumb navigation
- Article header with tag badge + read time
- Featured snippet box (highlighted intro)
- 4-6 content sections with H2 headings
- FAQ section with 4-5 Q&A pairs
- CTA box (gradient, signup link)
- Header + Footer components

## Files to Update After All Posts Created
1. **Blog index** (`src/app/(public)/blog/page.tsx`) - Add all 50 posts to the `posts` array
2. **Sitemap** (`src/app/sitemap.ts`) - Add all 50 blog URLs

## Execution Strategy
- Create posts in batches of 5 using parallel agent invocations
- Each agent creates 5 blog post page.tsx files
- After all 10 batches: update blog index + sitemap
- Final verification build
