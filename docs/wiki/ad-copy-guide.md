# Ad Copy Generator

Generate ad copy optimized for specific platforms with real-time character limit validation and multiple variations for A/B testing.

## How It Works

1. Select a **platform** — each has unique field requirements and character limits
2. Enter your **product details**, target audience, and key benefits
3. Choose a **tone** and number of variations
4. Set your **CTA goal** and optional landing page URL
5. Click **Generate** to create your ads
6. Edit generated copy inline with live character count validation
7. Export as CSV or JSON, or copy individually

## Supported Platforms

### Google Search
- Headlines (30 characters each)
- Descriptions (90 characters each)

### Google Display
- Short Headline (25 characters)
- Long Headline (90 characters)
- Description (90 characters)

### Meta (Facebook/Instagram)
- Primary Text (125 characters)
- Headline (40 characters)
- Description (30 characters)

### LinkedIn
- Intro Text (150 characters)
- Headline (70 characters)
- Description (100 characters)

### Twitter/X
- Tweet Copy (280 characters)
- Card Headline (70 characters)

### TikTok
- Ad Text (100 characters)
- Video Hook (50 characters)
- Video Body (200 characters)
- CTA (50 characters)

### Pinterest
- Title (100 characters)
- Description (500 characters)

## Character Limit Validation

Each generated ad field shows:
- **Green** — Well within the character limit
- **Yellow** — Approaching the limit (>90%)
- **Red** — Over the limit

Fields are editable after generation, and the character count updates in real-time. A progress bar below each field provides a visual indicator.

## Tips for Better Ad Copy

- **Be specific about benefits** — "Save 10 hours/week" beats "Save time"
- **Know your audience** — "Busy project managers at tech startups" produces more targeted copy than "professionals"
- **Set a clear CTA** — The CTA goal shapes the closing of each ad variation
- **Generate 3 variations** — A/B test different approaches to find what resonates
- **Edit after generation** — Use the generated copy as a starting point and refine for your brand voice

## Export Options

- **CSV** — Import directly into Google Ads, Meta Ads Manager, or spreadsheet tools
- **JSON** — Structured data for programmatic ad management
- **Copy All** — Full text of all variations to your clipboard
- **Copy Individual** — Copy a single variation or field
