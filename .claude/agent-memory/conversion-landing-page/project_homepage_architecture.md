name: Homepage Section Architecture

---

**Section order (approved):**
1. Hero
2. Trust bar (immediately below fold — first proof signal)
3. Testimonials (3 cards with star ratings, gradient avatars, role + company)
4. Differentiators (3-card grid, "Built different" badge)
5. How It Works (3-step numbered flow)
6. Supporting features grid (9 features, 3-col)
7. Final CTA (gradient card, dark background)

**Hero:**
- Badge: "AI Customer Support, Done Right"
- H1: Pain-led, references stopping repeated question answering, then outcome ("your knowledge base becomes a chatbot")
- Subheadline: Names three outcomes (deflect tickets, capture leads, book appointments) + deployment speed ("under an hour")
- Primary CTA: "Build Your Chatbot Free"
- Secondary CTA: "See Pricing"
- Trust micro-copy: "No credit card required — free plan available"

**Trust bar:**
- 4 signals: No credit card required, 14-day money-back guarantee, Deploy in under an hour, SOC 2-grade infrastructure
- Placed in a banded section with border-y for visual separation

**Testimonials:**
- 3 quotes; all from pricing page (reused); personas: Marketing Director/e-commerce, VP Customer Success/B2B SaaS, Founder/agency
- Section header: "Teams shipping faster with VocUI" — avoids "loved by" cliche

**Differentiators:**
- Each card: icon + badge label + outcome-led headline + explanation body
- Calendar booking, embeddable agent console, performance telemetry waterfall
- Section framing: "Most chatbot tools stop at Q&A. VocUI handles the full customer interaction loop."

**How It Works:**
- 3 numbered steps: Train → Customize → Deploy
- Desktop connector line between step numbers (purely decorative, aria-hidden)
- Ends with CTA: "Start Building Free"

**Supporting features (9):**
- Proactive Messaging (8 trigger types), Sentiment & Loyalty Scoring, Multi-Channel Deploy, Train on Anything, Authenticated Context, Lead Capture & Surveys, Support Ticketing, 20+ Languages, Full Widget Customization
- Framed as "the full platform behind it" — so it reads as depth, not bloat

**Final CTA:**
- Dark gradient card (primary-600 to primary-800)
- Headline calls out the cost of inaction: "Your support team is answering questions a chatbot could handle."
- Three outcome bullets in the sub-copy: deflect tickets, capture leads, book appointments
- Primary: "Build Your Chatbot Free" (white button), Secondary: "View Pricing" (outline-light)

**Why:** Trust bar before testimonials because logo/signal bars convert better before social proof cards. Differentiators before features grid so the compelling unique capabilities lead and the depth follows.
