name: CTA Copy (Locked)
description: Primary and secondary CTA language chosen for the homepage and the rationale behind each choice
type: project
---

**Primary CTA:** "Build Your Chatbot Free"
- Used in hero and final CTA section
- Verb-forward, names the action AND the value (it's free)
- Chosen over "Get Started Free" (too generic) and "Start Free Trial" (implies time limit)

**Secondary CTA (hero):** "See Pricing"
- Lower-commitment path for price-sensitive buyers
- Chosen over "View Demo" — the product doesn't have a hosted demo video

**Mid-page CTA (How It Works section):** "Start Building Free"
- Slight variation to keep the page from feeling repetitive
- Same value signal, slightly more active verb

**Final CTA section secondary:** "View Pricing"
- Matches the hero secondary for consistency

**Trust micro-copy near CTA:**
- "No credit card required — free plan available" (hero, below buttons)
- "No credit card required — 14-day money-back guarantee" (final CTA section, below buttons)

**Why:** Primary CTA tests better when it names both the action and the zero-cost entry point. "Build" is more specific than "Get Started" and hints at the product being a builder/creator tool.
