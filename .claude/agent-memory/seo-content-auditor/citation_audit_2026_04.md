Citation audit completed 2026-04-01: 6 blog posts audited and fixed for uncited statistical claims.

**Why:** Google's E-E-A-T framework requires authoritative sourcing for factual claims. Uncited statistics are a trustworthiness gap that can trigger HCU demotion.

**How to apply:** When adding new statistical claims to blog posts, always include an inline citation to the original source. Use the citation format: `<a href="URL" target="_blank" rel="noopener noreferrer" className="text-primary-600 dark:text-primary-400 hover:underline">Source Name</a>`. If no verifiable source exists, soften the language (e.g., "research suggests" instead of specific percentages).
